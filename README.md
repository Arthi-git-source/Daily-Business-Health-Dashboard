# Daily Business Health & Risk Dashboard

## Business Problem
Managers often realize revenue or operations issues too late. 
This project builds a daily dashboard to monitor key business KPIs 
and highlight early warning signals.

## Objective
- Track daily business performance
- Detect risks in revenue, operations, and customer experience
- Provide actionable insights for decision-makers

## Tools Used
- SQL
- Excel / Power BI
- Python (for data cleaning)

## Project Status
In progress
